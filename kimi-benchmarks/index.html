📝 Task Specification

Implement a mini LISP interpreter in Python supporting:

  • Data Types: integers, booleans (#t/#f), symbols, lists
  • Built-ins: +, -, *, /, =, <, >, cons, car, cdr, list, null?, number?, symbol?, list?
  • Special Forms: define, lambda, if, quote, cond
  • Features: lexical scoping with nested environments, recursion

Constraints: No eval(), no ast module, no parsing libraries. Build tokenizer, parser, and evaluator from scratch.
